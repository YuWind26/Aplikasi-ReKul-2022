# ReKul (Reminder Kuliah) merupakan sebuah aplikasi mobile dengan teknologi Kotlin yang memiliki fitur
<ul>
  <li>Register untuk Dosen</li>
  <li>Register untuk Mahasiswa</li>
  <li>Login untuk Dosen</li>
  <li>Login untuk Mahasiswa</li>
  <li>Home Admin</li>
  <li>Home Mahasiswa</li>
  <li>Tugas</li>
  <li>Jadwal</li>
  <li>Kontak</li>
  <li>Profil Dosen</li>
  <li>Profil Mahasiswa</li>
  <li>Edit Profil</li>
</ul>
